# Full-Stack-CAP-Application-with-Joule
Developing a Full-Stack CAP Application with Joule by https://developers.sap.com/tutorials

In this tutorial  used the Generative AI capabilities of Joule to create a CAP service in SAP Build Code
Prerequisites
You have an SAP BTP Trial account (Get a Free Account on SAP BTP Trial).
You have completed the Setup SAP Build Code tutorial https://github.com/ronasap/SAP-Build-Code-in-SAP-BTP/blob/be4a0b460d1252ba8246d3fe5b49e2f15577aa43/Setup%20SAP%20Build%20Code%20in%20SAP%20BTP.pdf.
You have created a Full Stack Node project as described in the Create an SAP Build Code Application from the Lobby tutorial
https://github.com/ronasap/Create-a-project-from-the-SAP-Build-lobby/blob/0ca959c1cce6aa7a714921de83f7df19a97aa954/Create%20a%20project%20from%20the%20lobby.pdf.
